# Opis projektu

## Cel projektu
Celem projektu jest stworzenie prostej aplikacji
do zarządzania codziennymi zadaniami.

## Problem
Użytkownicy mają trudność z organizacją obowiązków
i zapamiętywaniem terminów.

## Grupa docelowa
- studenci
- młodzi dorośli

## Zakres funkcjonalny
- dodawanie zadań
- przegląd listy zadań
- oznaczanie zadań jako wykonane
